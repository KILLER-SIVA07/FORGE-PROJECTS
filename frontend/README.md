# Frontend (Next.js)

Enterprise UI starter for the QR Visitor Management platform.

## Features Included
- Role-oriented views: Admin, Security, Staff, Login
- Shared navigation and KPI cards
- API client utilities for auth and events
- Dark mode, large-button gate console style

## Run
```bash
cd frontend
npm install
npm run dev
```

Set backend URL if needed:
```bash
export NEXT_PUBLIC_API_BASE=http://127.0.0.1:8000
```
